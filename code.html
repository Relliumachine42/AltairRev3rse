The Code

                    
                        function getValues() {
                            let userString = document.getElementById('userString').value;
                            displayString(reverseString(userString));
                        }
                        
                        function reverseString(input) {
                            let revString = '';
                            for (let i = input.length - 1; i >= 0; i--) {
                                let letter = input[i];
                                revString += letter;
                            }
                            return revString;
                        }
                        
                        function displayString(output) {
                            document.getElementById('results').textContent = output;
                            document.getElementById('alert').classList.remove('invisible');
                        }

The code is structured in three functions

REV3RSE

The first function getValues() pulls the string entered by the user into the input element with an ID of userString by initializing a variable with the value of that element. Then it gives the third function an argument consisting of the second function given an argument of the variable userString.

The second function reverseString() takes one parameter, corresponding to userString. It declares a variable revString with empty quotes so we have something to return. A for() loop counts backwards through the array that is the string provided by the user by starting with an index of one less than the array length. When that index is less than zero, it stops decrementing and exits the loop. Each loop, it assigns the content within the corresponding index of the array to a variable letter and adds that variable to revString. Upon exiting the loop, the function returns revString.

The third function displayString() takes the parameter corresponding to the output of the second function as established by the argument in the first function and puts that output as text content into the element with the ID results. Then it removes the class invisible from the element with the ID of alert, allowing the user to see the results.
